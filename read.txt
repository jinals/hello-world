please read below command when using git:

1. git init = for intitialization of your repository
2. git status = for knowing git status  to find out if there have been changes
3. git --version = to know version of the git software that is installed in your computer.
4. git config --global user.name "<Your Name>" = set your name
5. git config --global user.email "<youremail@example.com>" = set your email
6. git add readme.txt = add the file you just created you'd like to commit (aka save) to change.
7. git commit -m "your msg" = commit those changes to the repository's history with a short description of the updates.
